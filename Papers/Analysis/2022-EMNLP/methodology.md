# Methodology

## Overview
- The authors create **FLUTE**, a dataset of 9,000 figurative NLI instances with free-form textual explanations across four figurative language types: Sarcasm, Simile, Metaphor, and Idioms.
- A **model-in-the-loop framework** is used, combining GPT-3 (for generation), Amazon Mechanical Turk crowd workers (for minimal edits), and expert annotators (for quality control).
- The dataset is evaluated by fine-tuning a T5 model and comparing it against a baseline trained on e-SNLI using automatic metrics and human evaluation.

## Data Collection
- **Sarcasm**: 1,339 seed literal sentences from the Empathetic Dialogue dataset (Rashkin et al., 2019), selected from negative emotion categories (angry, afraid, embarrassed). GPT-3 generates literal paraphrases with emotion-conditioned prompting. Crowd workers convert paraphrases to sarcastic counterparts via minimal edits. Result: 1,339 entailment + 2,678 contradiction pairs.
- **Simile**: Sentences with similes extracted from Chakrabarty et al. (2021, 2022). GPT-3 generates literal paraphrases by replacing similes with their properties. Contradictions generated by inverting meaning w.r.t. the property. 108 challenging instances added from FigQA (Liu et al., 2022b). Result: 750 entailment + 750 contradiction pairs.
- **Metaphor**: 750 metaphors manually selected from Chakrabarty et al. (2021), Big-bench (Srivastava et al., 2022), and IMPLI (Stowe et al., 2022). GPT-3 generates paraphrases and contradictions that specifically target the metaphoric meaning (not minimal word edits). Result: 750 entailment + 750 contradiction pairs.
- **Idioms**: GPT-3 jointly generates paraphrases and contradictions using idiom meanings. Result: 1,000 entailment + 1,000 contradiction pairs.
- **Explanations**: GPT-3 generates free-form textual explanations for each <premise, hypothesis, label> tuple using few-shot prompting. For idioms, the idiom meaning is also provided in the prompt.

## System/Model Design
- **Joint Self-Rationalizing T5 Model**: Following Wiegreffe et al. (2021b), a T5 model is fine-tuned to jointly predict the label (Entails/Contradicts) and generate the explanation in a text-to-text format.
- **Instruction format**: "Does the sentence 'P' entail or contradict the sentence 'H'? Please answer between 'Entails' or 'Contradicts' and explain your decision in a sentence."
- **Two variants**:
  - **T5e-SNLI**: T5 (3B parameters) fine-tuned on e-SNLI (366,603 training examples) for 1 epoch, batch size 1024, AdamW optimizer, learning rate 1e-4. Neutral examples removed from e-SNLI.
  - **T5FLUTE**: T5 (3B parameters) fine-tuned on FLUTE (7,035 training examples, 50X smaller than e-SNLI) for 10 epochs, batch size 1024, AdamW optimizer, learning rate 1e-4, multitask across all four figurative language types.

## Algorithms and Techniques
- **Few-shot prompting with GPT-3**: Used for generating paraphrases, contradictions, and explanations. Task-specific prompts with examples are provided for each figurative language type.
- **Model-in-the-loop framework**: GPT-3 generates candidate instances → crowd workers perform minimal edits → experts verify and edit. This scales dataset creation while maintaining quality.
- **Expert verification protocol**: Three experts verify all GPT-3 outputs. Instances deemed insufficient are resampled (up to 3 rounds) or manually edited. Editing rates: sarcasm explanations 21%, simile 27%, metaphor 40%, idiom 10%.
- **Challenging instance selection**: For similes, instances where RoBERTa confuses entailment and contradiction are selected from FigQA to increase difficulty.

## Evaluation Process
- **Test set**: 1,500 instances — 750 from sarcasm, 250 each from simile, metaphor, and idiom.
- **Automatic metrics**:
  - **Explanation Score**: Average of BERTScore and BLEURT (0–100 scale).
  - **Accuracy@K**: Label accuracy at explanation score thresholds of 0, 50, and 60. Accuracy@0 = standard accuracy; Accuracy@50 counts only correct labels with explanation score > 50.
  - **Rationale Quality (RQ)**: Human simulatability variant — difference in accuracy between a model given input+rationale vs. input-only for gold label prediction (IR → Ô minus I → Ô). Computed using both predicted and gold rationales.
- **Human evaluation**:
  - 79 MTurk workers (≥98% HIT approval rate) evaluate 200 random samples (50 per figurative type).
  - Workers judge whether each explanation justifies the label (Yes/Weak Yes/Weak No/No) and identify shortcomings (Insufficient Justification, Too Trivial, Too Verbose, Untrue to Input, Violates Common Sense).
  - Inter-annotator agreement: Krippendorff's α = 0.45 (moderate agreement).
  - **Hscore**: Average human judgment score per explanation.
